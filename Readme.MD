 # 项目名称 : BongoCheat
<HR>

 ## 项目简介:
 该项目使用C++编写，可以修改BongoCat的点击次数：
 
 ## 文件结构:
 - main.cpp：主程序入口，负责调用各模块完成内存操作。
 - handle：提供获取进程ID和模块基址的功能。
 - address：计算目标内存地址的模块。
 


 ## 使用说明:
 - 变量“add_count” 表示：每次运行加多少点击次数，你可以自己修改。
 - 你可以在x64 Release模式下编译该项目，生成的可执行文件为BongoCheat.exe。
 - 如果不想编译，可以下载Realse 然后运行。
 - 运行时需要管理员权限。
